# 版本更新记录

## v1.3.0 (进行中)

### 修复
- 新号功能牌分配：仅给 T3 各 5 张，T0-T2 不给
- 新号默认上阵"交换师"和"洗牌师"

- 新手引导文案修正："蓝色"出牌 → 出牌；"详情"→"?"

### 新增
- 欢迎邮件道具：钻石 1888 + 金币 18888

- 服务端奖励计算（PvP + PvE）
- 称号 / 头像框系统（TITLES/FRAMES）
- 外装选择界面 showCosmeticSelect
- 段位表情与真实排行榜
- 全服邮件系统（发送 + 领取 + 红点）
- 分级 AI 难度系统
- 对战历史记录功能
- 商店 / 任务 / 对战奖励调整

### 优化
- 导航栏视觉升级，高度降至 66px
- 对局中隐藏导航栏
- 美术资源裁剪与卡片悬停效果
- 公用牌可读性优化
