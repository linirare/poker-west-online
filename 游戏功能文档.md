# 代号 001 — 游戏功能完整文档

> 西部酒馆扑克对战游戏，融合德州扑克牌型、卡牌技能、角色养成、天梯排位、PvP联机

## 一、核心玩法

### 对战规则

| 项目 | 说明 |
|------|------|
| 牌桌 | 标准52张扑克牌（♠♥♦♣，点数2-14） |
| 公共牌 | 每局固定3张，开局发出，全程可见 |
| 回合 | 最多5回合，5盘3胜制 |
| 手牌 | 每回合发满7张 |
| 出牌数 | 第1-3回合出3张，第4回合出4张，第5回合出5张 |
| 血量 | 每人3❤️，每输一回合扣1❤️ |
| 获胜条件 | 对方❤️归0 或 先赢3回合 |

### 牌型判定（从高到低）

| 排名 | 牌型 | 说明 |
|------|------|------|
| 9 | 皇家同花顺 | A-K-Q-J-10 同花色 |
| 8 | 同花顺 | 5张连续同花色 |
| 7 | 四条 | 4张相同点数 |
| 6 | 葫芦 | 3张相同 + 2张相同 |
| 5 | 同花 | 5张同花色 |
| 4 | 顺子 | 5张连续点数 |
| 3 | 三条 | 3张相同点数 |
| 2 | 两对 | 2组对子 |
| 1 | 一对 | 1组对子 |
| 0 | 高牌 | 无组合，比最大牌 |

### 对战流程

1. **准备阶段**：选2张功能牌、选难度（Easy/Medium/Hard）
2. **开局**：发3张公共牌，每人发7张手牌
3. **出牌阶段**：选择本回合要出的牌
4. **技能阶段**：可选择使用功能牌技能
5. **揭示**：双方亮牌，计算最佳5张牌型
6. **结算**：高牌型方赢（或平局），赢方对输方造成1❤️伤害
7. **下一回合**：补满手牌至7张，重复3-6
8. **结束**：一方❤️归0或3回合胜出

### AI难度

| 难度 | 技能使用率 | AI牌池 |
|------|-----------|--------|
| Easy (0) | 25% | 排除T0技能 |
| Medium (1) | 55% | 所有技能 |
| Hard (2) | 85% | 排除T3技能 |

## 二、技能系统

共 **22种功能牌**，4个稀有度等级：

### T0（最强 — 6种）

| 图标 | 名称 | 类型 | 效果 |
|------|------|------|------|
| 🃏 | 5-Joker | 策略 | 前4回合抽1张；第5回合抽Joker百搭 |
| 🎩 | 魔术师 | 复制 | 复制一张自己手牌 |
| 🕵️ | 神偷 | 偷取 | 偷一张对方手牌 |
| ✨ | 命运一击 | 转化 | 弃一张牌，获得同花色A |
| 💻 | 黑客 | 复制 | 复制对手一张手牌 |

### T1（高级 — 5种）

| 图标 | 名称 | 类型 | 效果 |
|------|------|------|------|
| 🍀 | 幸运日 | 增幅 | 本回合若打出顺子，视为同花顺 |
| 🧩 | 破坏专家 | 干扰 | 随机弃掉对手2张手牌 |
| 🧤 | 白手套 | 复制 | 复制一张公共牌到手牌 |
| ⏫ | 提前爆发 | 额外 | 本回合可以多打1张牌 |
| 🎲 | 赌神 | 抽牌 | 随机获得A/K/Q一张 |

### T2（中级 — 8种）

| 图标 | 名称 | 类型 | 效果 |
|------|------|------|------|
| ➕ | 加一减一 | 调整 | 选择一张手牌点数+1或-1 |
| 📋 | 情报员 | 选抽 | 额外抽2张，选1张加入手牌 |
| 🔍 | 我要验牌 | 信息 | 查看对方当前全部手牌 |
| ♻️ | 回收 | 回收 | 从弃牌记录取回1张曾打出的牌 |
| 🔒 | 封锁 | 干扰 | 对手本回合少打一张牌 |
| 🎲 | 幸运赌徒 | 随机 | 60%额外抽3张，否则弃1张手牌 |
| ♠️ | 同花信徒 | 预支 | 本回合若打出同花组合，下回合额外抽2张 |
| ⏳ | 拖延 | 干扰 | 令对手下回合少抽1张牌 |
| 🌀 | 扰乱 | 干扰 | 随机弃掉对方1张手牌 |

### T3（普通 — 3种）

| 图标 | 名称 | 类型 | 效果 |
|------|------|------|------|
| 🔀 | 洗牌师 | 置换 | 弃1张手牌，抽2张新牌 |
| 🔁 | 交换师 | 交换 | 选1张手牌与对方随机1张交换 |
| ⚡ | 蓄力 | 策略 | 本回合少打一张，下回合多打一张 |

### 技能规则

- 每局对战可**带入2张**功能牌
- 功能牌为**消耗品**：带入一局消耗1张库存
- 每回合可**使用1次**技能，技能使用后进入冷却（charge）
- 技能优先级：AI根据难度级别有不同使用策略

## 三、角色系统

共 **4个角色**，每个角色有被动经济加成：

| 图标 | 名称 | 稀有度 | 被动效果 |
|------|------|--------|----------|
| 🌟 | 牛仔 | 普通 | 金币收益 +5%起，每级额外+2% |
| 🛡️ | 守护队长 | 稀有 | 胜利额外钻石 +2💎起，每级+1💎 |
| 🎀 | 幸运莉莉 | 史诗 | 宝箱进度加速，每级+1进度 |
| 💎 | 宝石收藏家 | 传说 | 抽卡折扣 5%起，每级折扣+3%（最高30%） |

### 角色升级

- 升级消耗碎片 + 金币
- 碎片需求 = `8 + 当前等级 × 4`
- 金币需求 = 碎片数 × 80
- 每升1级，被动效果增强
- 新角色通过**抽卡**获得

## 四、天梯系统

共 **7个段位**：

| 段位 | 图标 | 晋级条件 | 保护机制 |
|------|------|----------|----------|
| 🌱 新手局 | 🌱 | 打3场或赢1场 | 前3场保护，不掉星 |
| 🥉 青铜 | 🥉 | 3星 | 赢+1星，输不掉星 |
| 🥈 白银 | 🥈 | 3星 | 赢+1星，输-1星 |
| 🥇 黄金 | 🥇 | 4星 | 赢+1星，输-1星 |
| 💎 铂金 | 💎 | 4星 | 赢+1星，输-1星 |
| 👑 大师 | 👑 | 积分制（1500分→传奇） | 胜+25分 / 负-15分 |
| 🌟 传奇 | 🌟 | - | 1500分以上进入 |

### 特殊机制

- **连胜奖励**：连胜≥3场且在铂金以下，额外+1星
- **降段保护**：新手局不掉星，青铜输不掉星
- **连败保护**：白银及以上输-1星，星数<0时降段

## 十、PvP联机对战

### 房间系统
- 基于Socket.IO实时通信
- 创建房间 / 加入房间 / 房间列表
- 房主可开始游戏或踢人
- 双方确认准备后开始对战

### PvP规则
- 与AI对战规则相同（5回合、3公共牌、牌型比较）
- 实时出牌、技能使用
- 聊天功能（房间内和全局）

### PvP奖励
- 胜利：200🪙 + 8💎
- 失败：60🪙 + 3💎
- 宝箱进度 +2（胜）/ +1（负）
- 计入战绩历史

## 十二、成就系统

记录在 `state.handCounts[10]` 中：

- **总览**：总局数、胜率、最大连胜、总金币、总钻石
- **牌型记录**：10种牌型各出现次数
- **技能使用记录**：每种功能牌使用次数
- **对战历史**：最近对局记录（胜负、比分、时间、模式）
